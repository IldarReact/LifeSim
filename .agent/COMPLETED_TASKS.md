## 1. ✅ Profit - знак исправлен

**Файл**: `features/activities/work/business-management.tsx`

**Изменение**:
```typescript
// Было:
value={`$${profit.toLocaleString()}/кв`}

// Стало:
value={`${profit >= 0 ? '+' : ''}$${profit.toLocaleString()}/кв`}
```

**Результат**: 
- Если прибыль положительная → показывается `+$1000/кв`
- Если прибыль отрицательная → показывается `-$500/кв`
- Если прибыль ноль → показывается `$0/кв`

## 2. ✅ Базовая работа из реальных вакансий

### Что сделано:

#### 2.1. Добавлена функция `getStartingJob()` 
**Файл**: `core/lib/data-loaders/jobs-loader.ts`

**Логика**:
1. Сначала ищет вакансии **без требований**
2. Если нет - ищет вакансии с **минимальными требованиями** (level 1)
3. В крайнем случае - возвращает первую доступную вакансию

```typescript
export function getStartingJob(
  countryId: string, 
  characterSkills: { id: string; level: number }[] = []
): Job | null
```

#### 2.2. Обновлен `createInitialPlayer()`
**Файл**: `core/lib/initialState.ts`

**Изменения**:
```typescript
// Было: создавалась фейковая вакансия
const initialJob: Job = {
  id: `job_${Date.now()}`,
  title: characterData.name,
  company: "Start Corp",
  // ...
}

// Стало: загружается реальная вакансия
```
