## Архитектура

### Поток Данных: Найм Онлайн-Игрока

```
1. EmployeeHireDialog → вкладка "Онлайн игроки"
   ↓
2. getOnlinePlayers() → список игроков
   ↓
3. Выбор игрока + настройка зарплаты/KPI
   ↓
4. createPlayerCandidate() → EmployeeCandidate с customSalary
   ↓
5. onHire(candidate) → hireEmployee в store
   ↓
6. business.employees[] += новый сотрудник (онлайн-игрок)
```

### Поток Данных: Открытие с Партнером

```
1. BusinessDetailDialog → кнопка "Открыть с партнером"
   ↓
2. PartnerSelectionDialog → выбор игрока
   ↓
3. Настройка доли владения (playerShare)
   ↓
4. onOpenWithPartner(partnerId, partnerName, playerShare)
   ↓
5. openBusinessWithPartner в store (TODO)
   ↓
6. business.partners[] += { player, npc: selectedPlayer }
```
